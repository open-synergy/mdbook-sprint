Method Call

  • API

Without Arg:

<host>/api/<str:model_name>/<int:id>/method/<str:method_name>/?token=<str:token>

With Arg:

<host>/api/<str:model_name>/<int:id>/method/<str:method_name>/?token=<str:token>&args=<list:id>

With Key Arg:

<host>/api/<str:model_name>/<int:id>/method/<str:method_name>/?token=<str:token>&kwargs=<dict:value>

Customer Invoice

Membuat Customer Invoice

Endpoint

Penjelasan lebih lanjut bisa dilihat disini

Parameter

  • model: account.invoice
  • create_vals:
KeyTypeDescription
internal_numberstringNomor Invoice
partner_idintegerID Partner yang didapat dari Mencari ID Partner
account_idintegerIsi dengan ID: 948
journal_idintegerIsi dengan ID: 46
typestringIsi dengan out_invoice
date_invoicedateTanggal Invoice
date_duedateTanggal Jatuh Tempo
invoice_line[(0, 0, <dict>)]Detail Invoices. Penjelasan ada dibawah
  • Penjelasan isi dictionary dari invoice_line:
KeyTypeDescription
namestringDeskripsi Invoice
account_idintegerID Account yang didapat dari Mencari ID Account
price_unitfloatHarga satuan
quantityfloatKuantitas
invoice_line_tax_id[(6, 0, <list_of_id>)]Apabila ada PPN, isi list_of_id dengan 3 jika exclude PPN, 4 jika include PPN. Apabila tidak ada hilangkan.

Contoh

create_vals="{
    'internal_number': 'TEST/INV/2020/10/10/0001',
    'partner_id': 22,
    'journal_id': 46,
    'account_id': 948,
    'type': 'out_invoice',
    'date_invoice': '2020-10-10',
    'date_due': '2020-10-10',
    'invoice_line': [
        (0, 0, {
            'name': 'Item #1',
            'account_id': 198,
            'price_unit': 40000.00,
            'quantity': 3,
            'invoice_line_tax_id': [(6, 0, [3])]
        }),
        (0, 0, {
            'name': 'Item #2',
            'account_id': 198,
            'price_unit': 100000.00,
            'quantity': 1,
        }),
    ]
}"
curl -i -d "token=858a5f372ea04cb7aad222fb6ba95f76&create_vals=${create_vals}" \
    https://demo8.simetri-sinergi.id/api/account.invoice/create

Attach Dokumen Pada Reimbursement

Endpoint

Penjelasan lebih lanjut bisa dilihat disini

Parameter

  • model:hr.reimbursement
  • create_vals:
KeyTypeDescription
namestringNama / Deskripsi dari Attachment
typestringIsi dengan "binary"
datasbase64File attachment yang sudah dikonversi menjadi format base64
datas_fnamestringNama File Attachment
res_modelstringIsikan dengan "hr.reimbursement"
res_idintegerID Reimbursement yang ingin dilampirkan attachment

Contoh

import requests
import base64

params = {
    "login": "admin",
    "password": "admin"
}
response = requests.get("http://localhost:8069/api/user/get_token?", params=params)
json_data = response.json()
token = json_data["token"]

datas = open(<full_path>, 'rb').read().encode('base64')


vals = {
    "name": "Test Attachment",
    "type": "binary",
    "datas": datas,
    "datas_fname": "file.txt",
    "res_model": "hr.reimbursement",
    "res_id": 8,
}

params = {
    "token": token,
    "create_vals": str(vals),
}

response = requests.get("http://localhost:8069/api/ir.attachment/create?", params=params)
print (response.content)
